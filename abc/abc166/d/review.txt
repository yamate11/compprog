2020/05/04
長時間
全探索/数学

1 <= X <= 1e9 なる整数 X に対し，A^5 - B^5 = X となる(正負の) 整数
A, Bを見つけよ．(XはA,Bが見つかるように取られている．)

「隣り合ったところが最小で，それはあまり大きくできない」と考えると
よかった．

A > B として良い．1e9 >= X = A^5 - B^5 >= A^5 - (A-1)^5
= (A-1 + 1)^5 - (A-1)^5 > 5*(A-1)^4 なので，A <= 200．同様に考えて
|A| <= 200, |B| <= 200 なので，400*400 程度を全探索すれば良い．
